## 💡 Descripción general

-   **Infraestructura de Red:** Cada nodo dispone de conectividad física mediante interfaz Gigabit Ethernet (1000BASE-T), garantizando una transferencia de datos de alta velocidad y baja latencia hacia el núcleo de red del centro.

-   **Directivas:** Los equipos operan bajo un entorno protegido mediante la implementación de directivas de grupo locales (GPO). Se han configurado objetos de directiva específicos en el editor de Directivas de Grupo (gpedit.msc) and Directivas de Seguridad Local (secpol.msc) para restringir el escalado de privilegios y asegurar la integridad del sistema operativo.

-   **Gestión de privilegios:** Para facilitar la flexibilidad docente, los usuarios mantienen permissions de administración; no obstante, el sistema implementa una capa de seguridad proactiva mediante directivas de red. Esto blinda los parámetros críticos del sistema y la configuración de red, evitando desajustes accidentales o la ejecución de software no autorizado, asegurando que el aula esté siempre operativa para la siguiente sesión, salvo vandalización del equipo.

-   **Navegación blindada:** Con el fin de proteger la privacidad de alumnos y docentes, todos los navegadores están configurados por defecto en [modo invitado](https://support.google.com/chrome/answer/6130773?hl=es&co=GENIE.Platform%3DDesktop), salvo Mozilla Firefox por no disponer de este modo. Este protocolo garantiza que, al cerrar la ventana, se eliminen automáticamente historiales, cookies y sesiones iniciadas. De esta forma, ningún dato personal o credencial queda expuesto para el siguiente usuario.
